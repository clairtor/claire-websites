# Elephants

A minimal Hugo theme using the Ubuntu font.

## Features
* No JavaScript
* High Contrast

## Installation
In the folder of your Hugo site, run:
```
    git submodule add https://gitlab.com/meibenny/elephants.git themes/elephants
```
